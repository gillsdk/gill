## Installation

Install `@gillsdk/solana-pay` with your package manager of choice:

```shell
npm install gill @gillsdk/solana-pay
```

```shell
pnpm add gill @gillsdk/solana-pay
```

```shell
yarn add gill @gillsdk/solana-pay
```
